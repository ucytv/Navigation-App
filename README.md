# Navigation-App

Navigation Component sayesinde Navigation View ile Bottom Navigation View araçlarını birlikte kullanabilirsiniz. Bu bileşen, basit ve işlevsel bir kod yapısına sahiptir.

You can use Navigation View and Bottom Navigation View tools together through Navigation Component. This component has a simple and functional code structure.
